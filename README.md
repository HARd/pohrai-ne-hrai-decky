# POHRAI/NE HRAI for Decky Loader

Decky Loader plugin that marks Ukrainian and hostile developers directly in the Steam Deck interface.

Install it in Decky:

```text
https://github.com/HARd/pohrai-ne-hrai-decky/releases/latest/download/pohrai-ne-hrai.zip
```

In Decky Loader, enable developer features, open Developer settings, choose Install Plugin from URL, and paste that URL.
